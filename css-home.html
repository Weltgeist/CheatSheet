CSS summary

This section is a summary view of various CSS tags, it provides details in HTML format.

HTML Table

The table below has a list of most common CSS tags and attribute:

Tag Code Description Type Categorie
Font family font-family The font-family property specifies the font for an element. Basic
Font size font-size The font-size property sets the size of a font. Basic
Font weight font-weight The font-weight property sets how thick or thin characters in text should be displayed. Basic
Text align text-align The text-align property specifies the horizontal alignment of text in an element. Basic
Vertical align vertical-align The vertical-align property sets the vertical alignment of an element. Basic
Color color The color property specifies the color of text. Basic
Background color background-color The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin). Basic
Opacity opacity The opacity property sets the opacity level for an element.The opacity-level describes the transparency-level, where 1 is not transparent at all, 0.5 is 50% see-through, and 0 is completely transparent. Basic
Background image background-image The background-image property sets one or more background images for an element.By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally. Basic
Url url() The url() CSS function is used to include a file Fonction Basic
Text transform text-transform The text-transform property controls the capitalization of text. Basic
